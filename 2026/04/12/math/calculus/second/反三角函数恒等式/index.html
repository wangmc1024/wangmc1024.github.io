二、x 与 1/x 时，也等于 π/2（最常考！）

1. 当 𝐱 > 𝟎

[
\arctan x + \arctan\frac{1}{x} = \frac{\pi}{2}
]

2. 当 𝐱 < 𝟎

[
\arctan x + \arctan\frac{1}{x} = -\frac{\pi}{2}
]

考研最爱考：看到 arctan A + arctan(1/A)
立刻反应：正 = π/2，负 = -π/2

三、什么时候 和 = π/2？总结成 3 条

  1. 同一个 x：
    arcsin + arccos、arctan + arccot、arcsec + arccsc
    ⇒ 必 = π/2

  2. x 与 1/x，且 x>0：
    arctan x + arctan(1/x) = π/2

  3. 互余角：若 α + β = π/2，则 arcsinα + arccosβ 之类也会出现 π/2
    （但考研基本只考前两类）

四、最简背诵版（考场直接用）

  • arcsin + arccos = π/2

  • arctan + arccot = π/2

  • x>0：arctan x + arctan(1/x) = π/2
