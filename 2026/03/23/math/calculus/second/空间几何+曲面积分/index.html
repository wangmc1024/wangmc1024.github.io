空间解析几何 + 曲面积分

（立体几何基础 → 曲面/交线 → 投影/求高 → 直接做题，全串起来）

一、上篇：空间解析几何（立体几何）——所有积分的基础工具

1. 向量代数（判断平行、垂直、求法向量）

  • 点积：(\vec{a}\cdot\vec{b}=x_1x_2+y_1y_2+z_1z_2)

    • (\vec{a}\perp\vec{b} \iff \vec{a}\cdot\vec{b}=0)

  • 叉积：(\vec{a}\times\vec{b}) → 求平面/直线法向量、方向向量

  • 混合积：((\vec{a}\times\vec{b})\cdot\vec{c}) → 判共面、求体积

2. 平面（曲面积分补面必用）

  • 点法式：(A(x-x_0)+B(y-y_0)+C(z-z_0)=0)
    (\vec{n}=(A,B,C)) 为法向量

  • 点到平面距离：
    [
    d=\frac{|Ax_0+By_0+Cz_0+D|}{\sqrt{A2+B2+C^2}}
    ]

3. 空间直线

  • 点向式：(\displaystyle\frac{x-x_0}{l}=\frac{y-y_0}{m}=\frac{z-z_0}{n})
    (\vec{s}=(l,m,n)) 为方向向量

  • 线面平行/垂直：看直线方向向量与平面法向量的关系

4. 基础曲面规则

  • 柱面：缺哪个变量，母线平行哪根轴例：(x2+y2=x)（无 (z)）→ 母线∥(z) 轴圆柱面

  • 旋转曲面：绕谁谁不变，另一变量变 (\sqrt{\cdot2+\cdot2})

二、下篇：曲面积分专用——曲面、交线、投影、求高（核心）

1. 真题必考曲面总表

曲面 方程 特征/积分用法
平面 (Ax+By+Cz=D) 高斯公式补面，构闭合曲面
球面 (x2+y2+z2=R2) 闭合曲面，直接高斯
圆柱面 (x2+y2=R^2) 母线∥(z) 轴，投影到 (xOz/yOz)
特殊圆柱 (x2+y2=x) (\left(x-\frac12\right)2+y2=\frac14)，(r=\cos\theta)
旋转抛物面 (z=x2+y2) 开口向上，常被 (z=h) 截断
圆锥面 (z=\sqrt{x2+y2}) 上半锥，顶点在原点
椭球面 (\frac{x2}{a2}+\frac{y2}{b2}+\frac{z2}{c2}=1) 闭合，高斯首选

    • 投 (yOz)：消 (x)，令 (x=0)

3. 圆锥面 / 旋转抛物面「求高」万能法（整合所有题型）

这里的高 = 交线z坐标 / 截断z值

（1）单个曲面被平面截

  • 旋转抛物面 (z=x2+y2) 被 (z=h) 截
    → 高 (=h)；若投影 (x2+y2\le R^2) → 高 (h=R^2)

  • 圆锥面 (z=\sqrt{x2+y2}) 被 (z=h) 截
    → 高 (=h)；若投影 (x2+y2\le R^2) → 高 (h=R)

（2）两曲面相交求高（通用3步）

  1. 联立方程

  2. 消去 (x2+y2)，只解 (z)

  3. 舍去 (z<0) 等不合理根

经典例子
球面 (x2+y2+z^2=2) + 抛物面 (z=x2+y2)
代入：(z+z^2=2\Rightarrow z=1)（舍去负根）
→ 交线高 (z=1)，投影 (x2+y2\le1)

4. 高频极坐标（直接背）

  • (x2+y2\le R^2)：(0\le r\le R,\ 0\le\theta\le2\pi)

  • (x2+y2=x)：(r=\cos\theta,\ \theta\in\left[-\frac\pi2,\frac\pi2\right])

  • (x2+y2=y)：(r=\sin\theta,\ \theta\in[0,\pi])
