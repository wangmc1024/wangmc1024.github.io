计算机主要性能指标

一、CPU 核心运算性能（408 高频考点）

1. 主频与时钟周期

  • 定义
    主频 ( f )：CPU 时钟脉冲频率，单位 ( \text{Hz、MHz、GHz} )，每秒有多少个时钟周期时钟周期 ( T )：一个时钟脉冲的时间，CPU 最小时间单位

  • 公式
    [
    T = \frac{1}{f}
    ]

  • 例题
    某 CPU 主频 ( f=3\ \text{GHz} )，求时钟周期。
    解：
    [
    T=\frac{1}{3\times10^9}\approx 3.33\times10^{-10}\ \text{s}=0.33\ \text{ns}
    ]

2. CPI（Clock cycles Per Instruction）

  • 定义：执行一条指令所需的平均时钟周期数

  • 公式
    [
    \text{CPI} = \frac{\text{总时钟周期数}}{\text{总指令数}}
    ]
    加权平均：
    [
    \text{CPI} = \sum_{i=1}^{n} \left( \text{CPI}_i \times \frac{\text{指令数}_i}{\text{总指令数}} \right)
    ]

  • 例题
    程序共 1000 条指令：

    • 400 条指令 CPI=2
    • 600 条指令 CPI=5
      求平均 CPI。
      解：
      [
      \text{CPI} = \frac{400\times2 + 600\times5}{1000} = 3.8
      ]

3. CPU 执行时间（必考公式）

  • 公式
    [
    \text{CPU 时间} = \text{指令数} \times \text{CPI} \times \text{时钟周期}
    ]
    或
    [
    \text{CPU 时间} = \frac{\text{指令数} \times \text{CPI}}{\text{主频}}
    ]

  • 例题
    指令数 ( 10^6 )，CPI=2，主频 ( 2\ \text{GHz} )，求执行时间。
    解：
    [
    \text{时间}=\frac{10^6 \times 2}{2\times109}=10{-3}\ \text{s}=1\ \text{ms}
    ]

4. IPC（Instructions Per Clock）

  • 定义：一个时钟周期执行的平均指令数

  • 公式
    [
    \text{IPC} = \frac{1}{\text{CPI}}
    ]

5. MIPS（百万条指令每秒）

  • 定义：每秒执行多少百万条指令

  • 公式
    [
    \text{MIPS} = \frac{\text{主频}}{\text{CPI}}
    ]
    或
    [
    \text{MIPS} = \frac{\text{指令数}}{\text{CPU 时间} \times 10^6}
    ]

  • 例题
    主频 4GHz，CPI=2，求 MIPS。
    解：
    [
    \text{MIPS}=\frac{4\times109}{2}=2\times109=2000\ \text{MIPS}
    ]

6. FLOPS（浮点运算速度）

  • 定义：每秒浮点运算次数，科学计算/AI 常用单位：MFLOPS、GFLOPS、TFLOPS

  • 公式
    [
    \text{GFLOPS} = \frac{\text{浮点运算总次数}}{\text{时间(s)} \times 10^9}
    ]

二、存储系统性能

1. 存储容量

  • 主存容量
    [
    \text{容量} = 2^{\text{地址线位数}} \times \text{字长}
    ]

  • 例题
    地址线 32 位，字长 32 位，求容量。
    解：
    [
    2^{32} \times 32\ \text{bit} = 4\ \text{GB} \times 4 = 16\ \text{GB}
    ]

2. 存储带宽

  • 定义：单位时间存储器可传输的数据量

  • 公式
    [
    \text{带宽(B/s)} = \text{数据总线宽度(bit)} \times \text{总线频率(Hz)} \div 8
    ]

  • 例题
    总线 64 位，频率 133MHz，求带宽。
    解：
    [
    \text{带宽} = \frac{64 \times 133\times10^6}{8} = 1064\ \text{MB/s}
    ]

3. 存取周期

  • 定义：连续两次独立访存操作的最小时间间隔，单位 ns
    存取周期 ≈ 存取时间 + 恢复时间

三、系统整体性能

  1. 吞吐量
    单位时间内系统完成的总任务数/总数据量，单位：任务/s、MB/s

  2. 响应时间
    从提交请求到得到结果的总时间
    = 等待时间 + 运行时间 + I/O 时间

四、408 常考综合题模板

已知：主频 ( f )、指令数 ( I )、各类指令 CPI 及占比求：CPI、CPU 时间、MIPS

  1. 先算加权平均 CPI

  2. 再算 ( \text{时间}=I\times\text{CPI}/f )

  3. 最后算 ( \text{MIPS}=f/\text{CPI} )
