多维随机变量及其分布知识点整理

一、多维随机变量的基本概念

1.1 多维随机变量的定义

设E是一个随机试验，样本空间为S = {e}。设X1 = X1(e), X2 = X2(e), …, Xn = Xn(e)是定义在S上的n个随机变量，由它们构成的一个n维向量(X1, X2, …, Xn)称为n维随机向量或n维随机变量。

1.2 联合分布函数

定义

对于n维随机变量(X1, X2, …, Xn)，n元函数： F(x1, x2, …, xn) = P{X1 ≤ x1, X2 ≤ x2, …, Xn ≤ xn} 称为n维随机变量(X1, X2, …, Xn)的联合分布函数。

二维联合分布函数

对于二维随机变量(X, Y)，联合分布函数为： F(x, y) = P{X ≤ x, Y ≤ y}

基本性质（高频考点）

  1. 单调性：F(x, y)是变量x和y的不减函数
  2. 有界性：0 ≤ F(x, y) ≤ 1
  3. 右连续性：F(x, y)关于x右连续，关于y右连续
  4. 边缘极限：
    • F(−∞, y) = 0
    • F(x, −∞) = 0
    • F(−∞, −∞) = 0
    • F(+∞, +∞) = 1
  5. 矩形不等式：对于任意x1 < x2, y1 < y2，有： F(x2, y2) − F(x1, y2) − F(x2, y1) + F(x1, y1) ≥ 0

常考题型：利用分布函数性质判断是否为 valid 的分布函数

二、二维离散型随机变量

2.1 联合概率分布律

定义

如果二维随机变量(X, Y)全部可能取到的不相同的值是有限对或可列无限多对，则称(X, Y)是离散型的二维随机变量。

设(X, Y)所有可能取值为(xi, yj)，i, j = 1, 2, …，则： P{X = xi, Y = yj} = pij,  i, j = 1, 2, … 称为二维离散型随机变量(X, Y)的联合概率分布律。

基本性质

  1. 非负性：pij ≥ 0
  2. 规范性：$\sum_{i=1}^{\infty} \sum_{j=1}^{\infty} p_{ij} = 1$

2.2 边缘分布律

定义

对于二维离散型随机变量(X, Y)： - X的边缘分布律：$p_i = P\{X = x_i\} = \sum_{j=1}^{\infty} p_{ij}$ - Y的边缘分布律：$p_j = P\{Y = y_j\} = \sum_{i=1}^{\infty} p_{ij}$

重要考点：联合分布可以确定边缘分布，但边缘分布不能确定联合分布

2.3 条件分布律

定义

设(X, Y)是二维离散型随机变量，对于固定的j，若P{Y = yj} > 0，则： $$P\{X = x_i | Y = y_j\} = \frac{P\{X = x_i, Y = y_j\}}{P\{Y = y_j\}} = \frac{p_{ij}}{p_j}, \quad i = 1, 2, \ldots$$ 称为在Y = yj条件下随机变量X的条件分布律。

同理，对于固定的i，若P{X = xi} > 0，则： $$P\{Y = y_j | X = x_i\} = \frac{P\{X = x_i, Y = y_j\}}{P\{X = x_i\}} = \frac{p_{ij}}{p_i}, \quad j = 1, 2, \ldots$$ 称为在X = xi条件下随机变量Y的条件分布律。

常考题型：由联合分布律求边缘分布律和条件分布律

三、二维连续型随机变量

3.1 联合概率密度函数

定义

设二维随机变量(X, Y)的分布函数为F(x, y)，如果存在非负可积的二元函数f(x, y)，使得对于任意实数x, y，有： F(x, y) = ∫−∞y∫−∞xf(u, v) du dv 则称(X, Y)是连续型的二维随机变量，函数f(x, y)称为二维随机变量(X, Y)的联合概率密度函数。

基本性质（高频考点）

  1. 非负性：f(x, y) ≥ 0
  2. 规范性：∫−∞∞∫−∞∞f(x, y) dx dy = 1
  3. 概率计算：P{(X, Y) ∈ G} = ∬Gf(x, y) dx dy
  4. 导数关系：在f(x, y)的连续点处，有$f(x, y) = \frac{\partial^2 F(x, y)}{\partial x \partial y}$

重要提醒：概率密度函数可以大于1，这与概率的取值范围不同

3.2 边缘概率密度

定义

对于二维连续型随机变量(X, Y)： - X的边缘概率密度：fX(x) = ∫−∞∞f(x, y) dy - Y的边缘概率密度：fY(y) = ∫−∞∞f(x, y) dx

计算要点： - 确定积分区域：根据联合概率密度的定义域确定积分上下限 - 分段函数处理：如果联合概率密度是分段函数，边缘概率密度也一定是分段函数

3.3 条件概率密度

定义

设二维连续型随机变量(X, Y)的联合概率密度为f(x, y)，边缘概率密度分别为fX(x)和fY(y)：

对于固定的y，若fY(y) > 0，则： $$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$$ 称为在Y = y条件下X的条件概率密度。

对于固定的x，若fX(x) > 0，则： $$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$$ 称为在X = x条件下Y的条件概率密度。

相应的条件分布函数为： $$F_{X|Y}(x|y) = \int_{-\infty}^{x} \frac{f(u, y)}{f_Y(y)} \, du$$ $$F_{Y|X}(y|x) = \int_{-\infty}^{y} \frac{f(x, v)}{f_X(x)} \, dv$$

常考题型： 1. 由联合概率密度求边缘概率密度 2. 计算条件概率密度 3. 利用条件概率密度求条件概率

四、随机变量的独立性

4.1 独立性定义

一般定义

设F(x, y)及FX(x)，FY(y)分别是二维随机变量(X, Y)的联合分布函数及边缘分布函数。若对于所有x, y有： F(x, y) = FX(x)FY(y) 则称随机变量X和Y是相互独立的。

离散型情况

对于二维离散型随机变量(X, Y)，X和Y相互独立的充要条件是： pij = pipj,  ∀i, j

连续型情况

对于二维连续型随机变量(X, Y)，X和Y相互独立的充要条件是： f(x, y) = fX(x)fY(y),  几乎处处成立

4.2 独立性的重要性质（高频考点）

  1. 函数独立性：若X和Y相互独立，则它们的函数g(X)和h(Y)也相互独立
  2. 多维推广：若X1, X2, …, Xn相互独立，则其中任意k个随机变量也相互独立
  3. 期望性质：若X和Y相互独立，则E[XY] = E[X]E[Y]
  4. 方差性质：若X和Y相互独立，则D(X + Y) = D(X) + D(Y)

4.3 不相关性

定义

若随机变量X和Y的相关系数ρXY = 0，则称X和Y不相关。

关系（重要考点）

  • 相互独立 ⟹ 不相关
  • 不相关 ⇏ 相互独立（除非是正态分布）

特殊情况：对于正态分布，独立性与不相关性是等价的

五、常用二维分布

5.1 二维均匀分布

定义

设G是平面上的有界区域，面积为A。若二维随机变量(X, Y)的联合概率密度为： $$f(x, y) = \begin{cases} \frac{1}{A}, & (x, y) \in G \\ 0, & \text{其他} \end{cases}$$ 则称(X, Y)在区域G上服从均匀分布。

性质

  1. 概率计算：$P\{(X, Y) \in D\} = \frac{\text{区域}D \cap G\text{的面积}}{A}$
  2. 边缘分布不一定是均匀分布

常考题型：在矩形区域、圆形区域等特殊区域上的均匀分布计算

5.2 二维正态分布（最重要的多维分布）

定义

若二维随机变量(X, Y)的联合概率密度为： $$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$ 其中μ1, μ2, σ1 > 0, σ2 > 0, |ρ| < 1是常数，则称(X, Y)服从参数为μ1, μ2, σ12, σ22, ρ的二维正态分布，记为(X, Y) ∼ N(μ1, μ2, σ12, σ22, ρ)。

参数意义

  • μ1, μ2：分别是X和Y的数学期望
  • σ12, σ22：分别是X和Y的方差
  • ρ：X和Y的相关系数

重要性质（高频考点）

  1. 边缘分布：X ∼ N(μ1, σ12)，Y ∼ N(μ2, σ22)
  2. 独立性：X和Y相互独立的充要条件是ρ = 0
  3. 条件分布：在给定Y = y条件下，X服从正态分布$N\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \sigma_1^2(1-\rho^2)\right)$
  4. 线性变换不变性：若(X, Y)服从二维正态分布，则对任意常数a, b, c, d，aX + bY和cX + dY也服从二维正态分布
  5. 线性组合正态性：若(X, Y)服从二维正态分布，则aX + bY服从一维正态分布N(aμ1 + bμ2, a2σ12 + b2σ22 + 2abρσ1σ2)

常考题型： 1. 二维正态分布的概率计算 2. 利用独立性条件（ρ = 0）求解问题 3. 条件分布的应用 4. 线性组合的分布计算

六、两个随机变量函数的分布

6.1 离散型情况

设(X, Y)是二维离散型随机变量，联合分布律为P{X = xi, Y = yj} = pij，Z = g(X, Y)，则Z的分布律为： $$P\{Z = z_k\} = \sum_{\substack{g(x_i, y_j) = z_k}} p_{ij}, \quad k = 1, 2, \ldots$$

6.2 连续型情况

设(X, Y)是二维连续型随机变量，联合概率密度为f(x, y)，Z = g(X, Y)，则Z的分布函数为： FZ(z) = P{Z ≤ z} = P{g(X, Y) ≤ z} = ∬g(x, y) ≤ zf(x, y) dx dy

Z的概率密度为： $$f_Z(z) = \frac{dF_Z(z)}{dz}$$

6.3 常用函数分布（重点掌握）

和的分布（卷积公式）

设Z = X + Y，则： fZ(z) = ∫−∞∞f(x, z − x) dx = ∫−∞∞f(z − y, y) dy

若X和Y相互独立，则： fZ(z) = fX * fY(z) = ∫−∞∞fX(x)fY(z − x) dx

重要特例： - 正态分布的可加性：若X ∼ N(μ1, σ12)，Y ∼ N(μ2, σ22)且独立，则X + Y ∼ N(μ1 + μ2, σ12 + σ22) - 泊松分布的可加性：若X ∼ P(λ1)，Y ∼ P(λ2)且独立，则X + Y ∼ P(λ1 + λ2) - 二项分布的可加性：若X ∼ B(n, p)，Y ∼ B(m, p)且独立，则X + Y ∼ B(n + m, p)

商的分布

设$Z = \frac{X}{Y}$，则： fZ(z) = ∫−∞∞|y|f(zy, y) dy

若X和Y相互独立，则： fZ(z) = ∫−∞∞|y|fX(zy)fY(y) dy

最大值和最小值分布（高频考点）

设M = max (X, Y)，N = min (X, Y)，X和Y相互独立：

FM(z) = FX(z)FY(z) FN(z) = 1 − [1 − FX(z)][1 − FY(z)]

推广到n个变量： 若X1, X2, …, Xn相互独立，M = max (X1, …, Xn)，N = min (X1, …, Xn)，则： $$F_M(z) = \prod_{i=1}^n F_{X_i}(z)$$ $$F_N(z) = 1 - \prod_{i=1}^n [1 - F_{X_i}(z)]$$

6.4 变换法（雅可比行列式法）

对于一一对应的变换$\begin{cases} Z = g(X, Y) \\ W = h(X, Y) \end{cases}$，联合概率密度为： fZ, W(z, w) = fX, Y(x(z, w), y(z, w))|J| 其中J是雅可比行列式： $$J = \begin{vmatrix} \frac{\partial x}{\partial z} & \frac{\partial x}{\partial w} \\ \frac{\partial y}{\partial z} & \frac{\partial y}{\partial w} \end{vmatrix}$$

常考题型： 1. 求两个随机变量和的分布 2. 求最大值、最小值的分布 3. 利用变换法求复杂函数的分布

七、重要公式总结

7.1 分布函数相关

  • 联合分布函数：F(x, y) = P{X ≤ x, Y ≤ y}
  • 边缘分布函数：FX(x) = F(x, +∞)，FY(y) = F(+∞, y)
  • 矩形概率：P{x1 < X ≤ x2, y1 < Y ≤ y2} = F(x2, y2) − F(x1, y2) − F(x2, y1) + F(x1, y1)

7.2 概率密度相关

  • 联合概率密度：$f(x, y) = \frac{\partial^2 F(x, y)}{\partial x \partial y}$
  • 边缘概率密度：fX(x) = ∫−∞∞f(x, y)dy，fY(y) = ∫−∞∞f(x, y)dx
  • 条件概率密度：$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$，$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$

7.3 独立性条件

  • 分布函数：F(x, y) = FX(x)FY(y)
  • 离散型：pij = pipj
  • 连续型：f(x, y) = fX(x)fY(y)

7.4 常用分布

  • 二维均匀分布：$f(x, y) = \frac{1}{A}, (x, y) \in G$
  • 二维正态分布：$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$

7.5 函数分布

  • 和的分布：fZ(z) = ∫−∞∞f(x, z − x)dx
  • 最大值分布：FM(z) = FX(z)FY(z)
  • 最小值分布：FN(z) = 1 − [1 − FX(z)][1 − FY(z)]

八、典型例题分析

例题1：联合分布律与边缘分布律（基础题型）

设二维离散型随机变量(X, Y)的联合分布律为：

Y 0 1 2
0 0.1 0.2 0.1
1 0.2 0.3 0.1

求：(1) X和Y的边缘分布律；(2) 判断X和Y是否独立。

解答： (1) 边缘分布律： - P{X = 0} = 0.1 + 0.2 = 0.3 - P{X = 1} = 0.2 + 0.3 = 0.5 - P{X = 2} = 0.1 + 0.1 = 0.2 - P{Y = 0} = 0.1 + 0.2 + 0.1 = 0.4 - P{Y = 1} = 0.2 + 0.3 + 0.1 = 0.6

  1. 独立性判断： P{X = 0, Y = 0} = 0.1，P{X = 0}P{Y = 0} = 0.3 × 0.4 = 0.12 由于0.1 ≠ 0.12，故X和Y不独立。

例题2：二维正态分布的性质（重要考点）

设(X, Y) ∼ N(1, 2, 4, 9, 0.5)，求：(1) X和Y的边缘分布；(2) P{X ≤ 1}；(3) 在Y = 2条件下X的条件分布。

解答： (1) 边缘分布： - X ∼ N(1, 4) - Y ∼ N(2, 9)

  1. $P\{X \leq 1\} = \Phi\left(\frac{1-1}{2}\right) = \Phi(0) = 0.5$

  2. 条件分布： $\mu_{X|Y=2} = 1 + 0.5 \times \frac{2}{3} \times (2-2) = 1$ σX|Y = 22 = 4 × (1 − 0.52) = 3 故X|Y = 2 ∼ N(1, 3)

例题3：两个随机变量函数的分布（高频考点）

设X和Y相互独立，且都服从N(0, 1)分布，求Z = X2 + Y2的概率密度。

解答： 使用分布函数法： FZ(z) = P{X2 + Y2 ≤ z}

当z ≤ 0时，FZ(z) = 0

当z > 0时，转换到极坐标： $$F_Z(z) = \int_{0}^{2\pi} \int_{0}^{\sqrt{z}} \frac{1}{2\pi} e^{-\frac{r^2}{2}} r \, dr \, d\theta = 1 - e^{-\frac{z}{2}}$$

因此，Z的概率密度为： $$f_Z(z) = \begin{cases} \frac{1}{2} e^{-\frac{z}{2}}, & z > 0 \\ 0, & z \leq 0 \end{cases}$$ 即Z ∼ χ2(2)分布（自由度为2的卡方分布）

例题4：最大值最小值分布（常考题型）

设X1, X2, …, Xn相互独立，且都服从U(0, 1)分布，求M = max (X1, X2, …, Xn)和N = min (X1, X2, …, Xn)的概率密度。

解答： 对于M = max (X1, …, Xn)： FM(z) = [FX(z)]n = zn,  0 < z < 1 fM(z) = nzn − 1,  0 < z < 1

对于N = min (X1, …, Xn)： FN(z) = 1 − [1 − FX(z)]n = 1 − (1 − z)n,  0 < z < 1 fN(z) = n(1 − z)n − 1,  0 < z < 1
