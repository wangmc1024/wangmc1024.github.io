多维随机变量及其分布知识点整理

一、多维随机变量的基本概念

1.1 多维随机变量的定义

设$E$是一个随机试验，样本空间为$S={e}$。设$X_1=X_1(e), X_2=X_2(e), \ldots, X_n=X_n(e)$是定义在$S$上的$n$个随机变量，由它们构成的一个$n$维向量$(X_1, X_2, \ldots, X_n)$称为$n$维随机向量或$n$维随机变量。

1.2 联合分布函数

定义

对于$n$维随机变量$(X_1, X_2, \ldots, X_n)$，$n$元函数：
$$F(x_1, x_2, \ldots, x_n) = P{X_1 \leq x_1, X_2 \leq x_2, \ldots, X_n \leq x_n}$$
称为$n$维随机变量$(X_1, X_2, \ldots, X_n)$的联合分布函数。

二维联合分布函数

对于二维随机变量$(X, Y)$，联合分布函数为：
$$F(x, y) = P{X \leq x, Y \leq y}$$

基本性质（高频考点）

  1. 单调性：$F(x, y)$是变量$x$和$y$的不减函数

  2. 有界性：$0 \leq F(x, y) \leq 1$

  3. 右连续性：$F(x, y)$关于$x$右连续，关于$y$右连续

  4. 边缘极限：

    • $F(-\infty, y) = 0$
    • $F(x, -\infty) = 0$
    • $F(-\infty, -\infty) = 0$
    • $F(+\infty, +\infty) = 1$

  5. 矩形不等式：对于任意$x_1 < x_2, y_1 < y_2$，有：
    $$F(x_2, y_2) - F(x_1, y_2) - F(x_2, y_1) + F(x_1, y_1) \geq 0$$

常考题型：利用分布函数性质判断是否为 valid 的分布函数

二、二维离散型随机变量

2.1 联合概率分布律

定义

如果二维随机变量$(X, Y)$全部可能取到的不相同的值是有限对或可列无限多对，则称$(X, Y)$是离散型的二维随机变量。

设$(X, Y)$所有可能取值为$(x_i, y_j)$，$i, j = 1, 2, \ldots$，则：
$$P{X = x_i, Y = y_j} = p_{ij}, \quad i, j = 1, 2, \ldots$$
称为二维离散型随机变量$(X, Y)$的联合概率分布律。

基本性质

  1. 非负性：$p_{ij} \geq 0$

  2. 规范性：$\sum_{i=1}^{\infty} \sum_{j=1}^{\infty} p_{ij} = 1$

2.2 边缘分布律

定义

对于二维离散型随机变量$(X, Y)$：

  • $X$的边缘分布律：$p_i = P{X = x_i} = \sum_{j=1}^{\infty} p_{ij}$

  • $Y$的边缘分布律：$p_j = P{Y = y_j} = \sum_{i=1}^{\infty} p_{ij}$

重要考点：联合分布可以确定边缘分布，但边缘分布不能确定联合分布

2.3 条件分布律

定义

设$(X, Y)$是二维离散型随机变量，对于固定的$j$，若$P{Y = y_j} > 0$，则：
$$P{X = x_i | Y = y_j} = \frac{P{X = x_i, Y = y_j}}{P{Y = y_j}} = \frac{p_{ij}}{p_j}, \quad i = 1, 2, \ldots$$
称为在$Y = y_j$条件下随机变量$X$的条件分布律。

同理，对于固定的$i$，若$P{X = x_i} > 0$，则：
$$P{Y = y_j | X = x_i} = \frac{P{X = x_i, Y = y_j}}{P{X = x_i}} = \frac{p_{ij}}{p_i}, \quad j = 1, 2, \ldots$$
称为在$X = x_i$条件下随机变量$Y$的条件分布律。

常考题型：由联合分布律求边缘分布律和条件分布律

三、二维连续型随机变量

3.1 联合概率密度函数

定义

设二维随机变量$(X, Y)$的分布函数为$F(x, y)$，如果存在非负可积的二元函数$f(x, y)$，使得对于任意实数$x, y$，有：
$$F(x, y) = \int_{-\infty}^{y} \int_{-\infty}^{x} f(u, v) , du , dv$$
则称$(X, Y)$是连续型的二维随机变量，函数$f(x, y)$称为二维随机变量$(X, Y)$的联合概率密度函数。

基本性质（高频考点）

  1. 非负性：$f(x, y) \geq 0$

  2. 规范性：$\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x, y) , dx , dy = 1$

  3. 概率计算：$P{(X, Y) \in G} = \iint_{G} f(x, y) , dx , dy$

  4. 导数关系：在$f(x, y)$的连续点处，有$f(x, y) = \frac{\partial^2 F(x, y)}{\partial x \partial y}$

重要提醒：概率密度函数可以大于1，这与概率的取值范围不同

3.2 边缘概率密度

定义

对于二维连续型随机变量$(X, Y)$：

  • $X$的边缘概率密度：$f_X(x) = \int_{-\infty}^{\infty} f(x, y) , dy$

  • $Y$的边缘概率密度：$f_Y(y) = \int_{-\infty}^{\infty} f(x, y) , dx$

计算要点：

  • 确定积分区域：根据联合概率密度的定义域确定积分上下限

  • 分段函数处理：如果联合概率密度是分段函数，边缘概率密度也一定是分段函数

3.3 条件概率密度

定义

设二维连续型随机变量$(X, Y)$的联合概率密度为$f(x, y)$，边缘概率密度分别为$f_X(x)$和$f_Y(y)$：

对于固定的$y$，若$f_Y(y) > 0$，则：
$$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$$
称为在$Y = y$条件下$X$的条件概率密度。

对于固定的$x$，若$f_X(x) > 0$，则：
$$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$$
称为在$X = x$条件下$Y$的条件概率密度。

相应的条件分布函数为：
$$F_{X|Y}(x|y) = \int_{-\infty}^{x} \frac{f(u, y)}{f_Y(y)} , du$$
$$F_{Y|X}(y|x) = \int_{-\infty}^{y} \frac{f(x, v)}{f_X(x)} , dv$$

常考题型：

  1. 由联合概率密度求边缘概率密度

  2. 计算条件概率密度

  3. 利用条件概率密度求条件概率

四、随机变量的独立性

4.1 独立性定义

一般定义

设$F(x, y)$及$F_X(x)$，$F_Y(y)$分别是二维随机变量$(X, Y)$的联合分布函数及边缘分布函数。若对于所有$x, y$有：
$$F(x, y) = F_X(x) F_Y(y)$$
则称随机变量$X$和$Y$是相互独立的。

离散型情况

对于二维离散型随机变量$(X, Y)$，$X$和$Y$相互独立的充要条件是：
$$p_{ij} = p_i p_j, \quad \forall i, j$$

连续型情况

对于二维连续型随机变量$(X, Y)$，$X$和$Y$相互独立的充要条件是：
$$f(x, y) = f_X(x) f_Y(y), \quad \text{几乎处处成立}$$

4.2 独立性的重要性质（高频考点）

  1. 函数独立性：若$X$和$Y$相互独立，则它们的函数$g(X)$和$h(Y)$也相互独立

  2. 多维推广：若$X_1, X_2, \ldots, X_n$相互独立，则其中任意$k$个随机变量也相互独立

  3. 期望性质：若$X$和$Y$相互独立，则$E[XY] = E[X]E[Y]$

  4. 方差性质：若$X$和$Y$相互独立，则$D(X + Y) = D(X) + D(Y)$

4.3 不相关性

定义

若随机变量$X$和$Y$的相关系数$\rho_{XY} = 0$，则称$X$和$Y$不相关。

关系（重要考点）

  • 相互独立 $\implies$ 不相关

  • 不相关 $\nRightarrow$ 相互独立（除非是正态分布）

特殊情况：对于正态分布，独立性与不相关性是等价的

五、常用二维分布

5.1 二维均匀分布

定义

设$G$是平面上的有界区域，面积为$A$。若二维随机变量$(X, Y)$的联合概率密度为：
$$f(x, y) = \begin{cases}
\frac{1}{A}, & (x, y) \in G \
0, & \text{其他}
\end{cases}$$
则称$(X, Y)$在区域$G$上服从均匀分布。

性质

  1. 概率计算：$P{(X, Y) \in D} = \frac{\text{区域}D \cap G\text{的面积}}{A}$

  2. 边缘分布不一定是均匀分布

常考题型：在矩形区域、圆形区域等特殊区域上的均匀分布计算

5.2 二维正态分布（最重要的多维分布）

定义

若二维随机变量$(X, Y)$的联合概率密度为：
$$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left{-\frac{1}{2(1-\rho2)}\left[\frac{(x-\mu_1)2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)2}{\sigma_22}\right]\right}$$
其中$\mu_1, \mu_2, \sigma_1 > 0, \sigma_2 > 0, |\rho| < 1$是常数，则称$(X, Y)$服从参数为$\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho$的二维正态分布，记为$(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$。

参数意义

  • $\mu_1, \mu_2$：分别是$X$和$Y$的数学期望

  • $\sigma_1^2, \sigma_2^2$：分别是$X$和$Y$的方差

  • $\rho$：$X$和$Y$的相关系数

重要性质（高频考点）

  1. 边缘分布：$X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$

  2. 独立性：$X$和$Y$相互独立的充要条件是$\rho = 0$

  3. 条件分布：在给定$Y = y$条件下，$X$服从正态分布$N\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \sigma_12(1-\rho2)\right)$

  4. 线性变换不变性：若$(X, Y)$服从二维正态分布，则对任意常数$a, b, c, d$，$aX + bY$和$cX + dY$也服从二维正态分布

  5. 线性组合正态性：若$(X, Y)$服从二维正态分布，则$aX + bY$服从一维正态分布$N(a\mu_1 + b\mu_2, a2\sigma_12 + b2\sigma_22 + 2ab\rho\sigma_1\sigma_2)$

常考题型：

  1. 二维正态分布的概率计算

  2. 利用独立性条件（$\rho = 0$）求解问题

  3. 条件分布的应用

  4. 线性组合的分布计算

六、两个随机变量函数的分布

6.1 离散型情况

设$(X, Y)$是二维离散型随机变量，联合分布律为$P{X = x_i, Y = y_j} = p_{ij}$，$Z = g(X, Y)$，则$Z$的分布律为：
$$P{Z = z_k} = \sum_{\substack{g(x_i, y_j) = z_k}} p_{ij}, \quad k = 1, 2, \ldots$$

6.2 连续型情况

设$(X, Y)$是二维连续型随机变量，联合概率密度为$f(x, y)$，$Z = g(X, Y)$，则$Z$的分布函数为：
$$F_Z(z) = P{Z \leq z} = P{g(X, Y) \leq z} = \iint_{g(x, y) \leq z} f(x, y) , dx , dy$$

$Z$的概率密度为：
$$f_Z(z) = \frac{dF_Z(z)}{dz}$$

6.3 常用函数分布（重点掌握）

和的分布（卷积公式）

设$Z = X + Y$，则：
$$f_Z(z) = \int_{-\infty}^{\infty} f(x, z - x) , dx = \int_{-\infty}^{\infty} f(z - y, y) , dy$$

若$X$和$Y$相互独立，则：
$$f_Z(z) = f_X * f_Y(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z - x) , dx$$

重要特例：

  • 正态分布的可加性：若$X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$且独立，则$X + Y \sim N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$

  • 泊松分布的可加性：若$X \sim P(\lambda_1)$，$Y \sim P(\lambda_2)$且独立，则$X + Y \sim P(\lambda_1 + \lambda_2)$

  • 二项分布的可加性：若$X \sim B(n, p)$，$Y \sim B(m, p)$且独立，则$X + Y \sim B(n + m, p)$

商的分布

设$Z = \frac{X}{Y}$，则：
$$f_Z(z) = \int_{-\infty}^{\infty} |y| f(zy, y) , dy$$

若$X$和$Y$相互独立，则：
$$f_Z(z) = \int_{-\infty}^{\infty} |y| f_X(zy) f_Y(y) , dy$$

最大值和最小值分布（高频考点）

设$M = \max(X, Y)$，$N = \min(X, Y)$，$X$和$Y$相互独立：

$$F_M(z) = F_X(z) F_Y(z)$$
$$F_N(z) = 1 - [1 - F_X(z)][1 - F_Y(z)]$$

推广到n个变量：若$X_1, X_2, \ldots, X_n$相互独立，$M = \max(X_1, \ldots, X_n)$，$N = \min(X_1, \ldots, X_n)$，则：
$$F_M(z) = \prod_{i=1}^n F_{X_i}(z)$$
$$F_N(z) = 1 - \prod_{i=1}^n [1 - F_{X_i}(z)]$$

6.4 变换法（雅可比行列式法）

对于一一对应的变换$\begin{cases} Z = g(X, Y) \ W = h(X, Y) \end{cases}$，联合概率密度为：
$$f_{Z,W}(z, w) = f_{X,Y}(x(z,w), y(z,w)) |J|$$
其中$J$是雅可比行列式：
$$J = \begin{vmatrix} \frac{\partial x}{\partial z} & \frac{\partial x}{\partial w} \ \frac{\partial y}{\partial z} & \frac{\partial y}{\partial w} \end{vmatrix}$$

常考题型：

  1. 求两个随机变量和的分布

  2. 求最大值、最小值的分布

  3. 利用变换法求复杂函数的分布

七、重要公式总结

7.1 分布函数相关

  • 联合分布函数：$F(x, y) = P{X \leq x, Y \leq y}$

  • 边缘分布函数：$F_X(x) = F(x, +\infty)$，$F_Y(y) = F(+\infty, y)$

  • 矩形概率：$P{x_1 < X \leq x_2, y_1 < Y \leq y_2} = F(x_2,y_2) - F(x_1,y_2) - F(x_2,y_1) + F(x_1,y_1)$

7.2 概率密度相关

  • 联合概率密度：$f(x, y) = \frac{\partial^2 F(x, y)}{\partial x \partial y}$

  • 边缘概率密度：$f_X(x) = \int_{-\infty}^{\infty} f(x, y) dy$，$f_Y(y) = \int_{-\infty}^{\infty} f(x, y) dx$

  • 条件概率密度：$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$，$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$

7.3 独立性条件

  • 分布函数：$F(x, y) = F_X(x)F_Y(y)$

  • 离散型：$p_{ij} = p_i p_j$

  • 连续型：$f(x, y) = f_X(x)f_Y(y)$

7.4 常用分布

  • 二维均匀分布：$f(x, y) = \frac{1}{A}, (x, y) \in G$

  • 二维正态分布：$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left{-\frac{1}{2(1-\rho2)}\left[\frac{(x-\mu_1)2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)2}{\sigma_22}\right]\right}$

7.5 函数分布

  • 和的分布：$f_Z(z) = \int_{-\infty}^{\infty} f(x, z-x) dx$

  • 最大值分布：$F_M(z) = F_X(z)F_Y(z)$

  • 最小值分布：$F_N(z) = 1 - [1-F_X(z)][1-F_Y(z)]$

八、典型例题分析

例题1：联合分布律与边缘分布律（基础题型）

设二维离散型随机变量$(X, Y)$的联合分布律为：

Y\X 0 1 2
0 0.1 0.2 0.1
1 0.2 0.3 0.1

求：(1) $X$和$Y$的边缘分布律；(2) 判断$X$和$Y$是否独立。

解答：
(1) 边缘分布律：

  • $P{X=0} = 0.1 + 0.2 = 0.3$

  • $P{X=1} = 0.2 + 0.3 = 0.5$

  • $P{X=2} = 0.1 + 0.1 = 0.2$

  • $P{Y=0} = 0.1 + 0.2 + 0.1 = 0.4$

  • $P{Y=1} = 0.2 + 0.3 + 0.1 = 0.6$

(2) 独立性判断：
$P{X=0,Y=0} = 0.1$，$P{X=0}P{Y=0} = 0.3 \times 0.4 = 0.12$
由于$0.1 \neq 0.12$，故$X$和$Y$不独立。

例题2：二维正态分布的性质（重要考点）

设$(X, Y) \sim N(1, 2, 4, 9, 0.5)$，求：(1) $X$和$Y$的边缘分布；(2) $P{X \leq 1}$；(3) 在$Y = 2$条件下$X$的条件分布。

解答：
(1) 边缘分布：

  • $X \sim N(1, 4)$

  • $Y \sim N(2, 9)$

(2) $P{X \leq 1} = \Phi\left(\frac{1-1}{2}\right) = \Phi(0) = 0.5$

(3) 条件分布：
$\mu_{X|Y=2} = 1 + 0.5 \times \frac{2}{3} \times (2-2) = 1$
$\sigma_{X|Y=2}^2 = 4 \times (1 - 0.5^2) = 3$
故$X|Y=2 \sim N(1, 3)$

例题3：两个随机变量函数的分布（高频考点）

设$X$和$Y$相互独立，且都服从$N(0, 1)$分布，求$Z = X^2 + Y^2$的概率密度。

解答：
使用分布函数法：
$$F_Z(z) = P{X^2 + Y^2 \leq z}$$

当$z \leq 0$时，$F_Z(z) = 0$

当$z > 0$时，转换到极坐标：
$$F_Z(z) = \int_{0}^{2\pi} \int_{0}^{\sqrt{z}} \frac{1}{2\pi} e{-\frac{r2}{2}} r , dr , d\theta = 1 - e^{-\frac{z}{2}}$$

因此，$Z$的概率密度为：
$$f_Z(z) = \begin{cases}
\frac{1}{2} e^{-\frac{z}{2}}, & z > 0 \
0, & z \leq 0
\end{cases}$$
即$Z \sim \chi^2(2)$分布（自由度为2的卡方分布）

例题4：最大值最小值分布（常考题型）

设$X_1, X_2, \ldots, X_n$相互独立，且都服从$U(0, 1)$分布，求$M = \max(X_1, X_2, \ldots, X_n)$和$N = \min(X_1, X_2, \ldots, X_n)$的概率密度。

解答：
对于$M = \max(X_1, \ldots, X_n)$：
$$F_M(z) = [F_X(z)]^n = z^n, \quad 0 < z < 1$$
$$f_M(z) = n z^{n-1}, \quad 0 < z < 1$$

对于$N = \min(X_1, \ldots, X_n)$：
$$F_N(z) = 1 - [1 - F_X(z)]^n = 1 - (1 - z)^n, \quad 0 < z < 1$$
$$f_N(z) = n (1 - z)^{n-1}, \quad 0 < z < 1$$
