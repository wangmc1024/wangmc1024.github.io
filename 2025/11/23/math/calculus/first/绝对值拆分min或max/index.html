题目1：连续性证明（基础应用）

题干：设函数g(x)在ℝ上连续，令g+(x) = max {g(x), 1}，证明g+(x)在ℝ上连续。 解题步骤： 1. 公式变形：先将max {g(x), 1}转化为可利用绝对值拆分的形式，令h(x) = g(x) − 1，则max {g(x), 1} = max {h(x), 0} + 1。 由正部拆分公式，代入得： 2. 连续性分析： - g(x)连续 ⟹ g(x) − 1连续 ⟹ |g(x) − 1|连续（连续函数的复合运算）； - |g(x) − 1|+g(x) + 1是连续函数的和，仍连续； - 数乘后仍连续，故g+(x)连续。

题目2：函数表达式化简（公式直接套用）

题干：已知f(x) = 2x − 3，利用公式写出f+(x) = max {f(x), 0}和f−(x) = −min {f(x), 0}的具体表达式，并化简。 解题步骤：

  1. 由正部拆分公式，代入f(x) = 2x − 3得： 分情况化简：

题目3：函数性质分析（组合还原公式应用）

题干：设f(x)是定义在ℝ上的函数，f+(x) = max {f(x), 0}，f−(x) = −min {f(x), 0}，证明：|f(x)| = f+(x) + f−(x)。 解题步骤： 1. 由正部拆分公式得； 由负部拆分公式得； 2. 两式相加： 同时结合组合还原公式f(x) = f+(x) − min {f(x), 0} = f+(x) − (−f−(x)) = f+(x) + f−(x)，可进一步推导f(x)与|f(x)|的关系。
