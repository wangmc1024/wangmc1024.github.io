间断点在函数四则,复合运算中的变化

发表于 2025-11-23 更新于 2025-11-26 分类于 考研 阅读次数：
本文字数： 2.1k 阅读时长 ≈ 2 分钟

当两个函数 f(x) 和 g(x) 拥有不同的间断点时，它们进行运算后，新函数的连续性并不只是简单的“叠加”，而是存在“传染”、“屏蔽”、“修复”与“新生”四种复杂关系。

我们假设 x0 是考察点。

第一部分：四则运算中的连续性博弈

法则核心： 连续性是一个“好”性质。

  1. 好 + 好 = 好 （闭合运算）
  2. 好 + 坏 = 坏 （必然传染）
  3. 坏 + 坏 = 不一定 （可能负负得正）

1. 加减法运算 (f ± g)

假设 x0 是 f(x) 的间断点，而是 g(x) 的连续点。

  • 结论： f(x) ± g(x) 在 x0 处必为间断点。

  • 原理（反证法）：

    假设 h(x) = f(x) + g(x) 在 x0 连续。

    因为 g(x) 连续，根据“连续函数的差还是连续函数”，那么 f(x) = h(x) − g(x) 也必须是连续的。这与“f(x) 在 x0 间断”矛盾。

  • 通俗理解： 一个人平稳走路（连续），另一个人跳了一下（间断）。把他们俩绑在一起走（相加），整体一定会跳一下。

特殊情况： 如果 f和 g 在 x0 都是间断点，它们的和可能变成连续函数。

  • 例子： f(x) = sgn(x) （在0处跳跃），g(x) = −sgn(x)。
  • f + g = 0 （常数函数，恒连续）。间断点互相抵消了。

2. 乘法运算 (f ⋅ g) —— 最容易出错的地方

假设 x0 是 f(x) 的间断点，而是 g(x) 的连续点。

  • 结论： f(x) ⋅ g(x) 在 x0 不一定是间断点。

  • 关键因子： 取决于 g(x0) 是否等于 0。

    • 情形 A： g(x0) ≠ 0。

      此时“坏基因”被保留，积函数在 x0 必间断。

    • 情形 B： g(x0) = 0 （零的屏蔽效应）。

      如果 f(x) 在 x0 处是有界间断（比如震荡或跳跃），g(x) 的趋于0可以将这个不连续“压平”为连续。

  • 经典案例：

    • 在 x = 0 处剧烈震荡（第二类间断点）。
    • g(x) = x 在 x = 0 处连续且为0。
    • 。当 x → 0 时，极限为0，补充定义 h(0) = 0 后，该函数在 x = 0 变连续了。
    • 解释： g(x) 像一个强大的平滑剂，抹平了 f(x) 的躁动。

3. 除法运算 (f/g) —— 制造麻烦的机器

除法不仅继承 f 和 g 原有的间断点，还会制造新的间断点。

  • 继承性： 分子或分母原有的间断点，通常会保留下来。

  • 新生性（分母为零）：

    即使 f 和 g 在某处都连续，只要 g(x0) = 0，这一点就会变成新函数的间断点（通常是无穷间断点）。

第二部分：复合运算中的连续性传递

对于 y = f(g(x))，连续性的传递像是一个“接力赛”。

逻辑链：。

1. 第一关断裂：内层 g(x) 在 x0 间断

  • 一般情况： 复合函数 f(g(x)) 在 x0 也是间断的。

  • 例外（修复效应）：

    如果外层函数 f(u) 是一个常数函数，或者 f(u) 具有某种对称性将 g(x) 的跳跃“粘合”了。

    • 案例： g(x) = 1(x > 0), −1(x ≤ 0)，在0处跳跃。
    • 外层 f(u) = u2。
    • f(g(x))：无论是1还是-1，平方后都是1。新函数 y = 1 恒连续。外层函数的偶性修复了内层的跳跃间断点。

2. 第二关断裂：内层 g(x) 连续，但外层 f(u) 在 u0 = g(x0) 处间断

这是复合函数最隐蔽的间断点来源。

  • 现象： x0 原本对于 g 来说是正常的，但它产出的结果 u0 恰好打在了 f 的软肋（间断点）上。
  • 结论： x0 会成为复合函数的间断点。
  • 案例推导：
    • 内层 g(x) = x2 − 1 （处处连续）。
    • 外层 （在 u = 0 处间断）。
    • 新间断点： 令 g(x) = 0 ⇒ x2 − 1 = 0 ⇒ x = ±1。
    • 虽然 ±1 对 g 没问题，但对 f 来说是致命的。所以 x = ±1 是复合函数的无穷间断点。

第三部分：总结表与判断逻辑

假设 f(x) 在 x0 间断，g(x) 在 x0 连续。

运算类型 新函数在 x0 的状态 核心判定理由与特殊情况
加减 f ± g 必间断 连续 + 间断 = 间断（绝对铁律）。
乘法 f ⋅ g 不一定 若 g(x0) ≠ 0，必间断。 若 g(x0) = 0 且 f 有界，可能变为连续（零因子屏蔽）。
除法 f/g 必间断 分子间断或者分母为0都会导致间断。
复合 f(g(x)) 视情况定 若 g 连续但 g(x0) 落在 f 的间断点上 → 间断。 若 g 间断但 f 将其值域映射为单点 → 可能连续。

第四部分：深度解析——间断点的类型变化

如果两个函数运算后依然间断，间断点的类型会变吗？

  1. 可去间断点的“治愈”：
    • f(x) = x，（在 x = 1 处是可去间断点）。
    • 如果做除法 （定义域仍需排除0和1，但极限存在）。通过代数化简，有时看起来像“治愈”了，但严格定义上定义域仍有空洞，需补充定义才算连续。
  2. 无穷间断点的“爆发”：
    • 两个连续函数相除，分母趋近于0的速度如果快于分子（或分子不为0），就会把“连续”变成“无穷间断”。
  3. 跳跃间断点的“平滑化”：
    • 如前所述，利用 x2 或 |x| 等偶函数包裹一个跳跃函数，可以将“跳跃间断点”转化为“连续点”或“可去间断点”。

总结

分析新函数的连续性时，请遵循以下步骤：

  1. 找疑点： 罗列 f 的间断点、g 的间断点、以及 g(x) = 0 的点。
  2. 看加减： 只要一个坏一个好，结果必坏。
  3. 看乘法： 遇到连续函数值为 0 的点，要警惕，它可能会把间断点“救活”。
  4. 看复合： 跟踪 x 的轨迹，看它经过 g 变换后的值，是否落入了 f 的陷阱里。
