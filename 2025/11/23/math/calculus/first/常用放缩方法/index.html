2. 积分的放缩技巧

用于积分不等式证明、定积分估值，核心是利用“被积函数的单调性”或“积分区间的包含关系”放缩。 #### （1）被积函数单调性放缩 若在区间[a, b]上，f(x) ≤ g(x)，则∫abf(x)dx ≤ ∫abg(x)dx； 若f(x)在[a, b]上单调递增，则f(a)(b − a) ≤ ∫abf(x)dx ≤ f(b)(b − a)。 例：估计∫01exdx的范围 f(x) = ex在[0, 1]单调递增，故e0(1 − 0) ≤ ∫01exdx ≤ e1(1 − 0)，即1 ≤ 积分 ≤ e。

（2）积分区间分割与绝对值放缩

  • 区间分割：若a < c < b，则∫abf(x)dx = ∫acf(x)dx + ∫cbf(x)dx，可对分段区间分别放缩；
  • 绝对值积分不等式：|∫abf(x)dx| ≤ ∫ab|f(x)|dx，适用于含绝对值的积分估计。

3. 级数的放缩技巧

主要用于正项级数敛散性的比较审敛法，核心是将目标级数放缩为“已知敛散性的级数（p级数、几何级数）”。 #### （1）正项级数的放缩准则 若和为正项级数，且un ≤ vn： - 若∑vn收敛，则∑un收敛； - 若∑un发散，则∑vn发散。 常用放缩目标： - p级数：，p > 1收敛，p ≤ 1发散； - 几何级数：，|r| < 1收敛，|r| ≥ 1发散。

二、线性代数中的放缩技巧（低频考点，侧重矩阵/特征值）

线性代数的放缩主要用于矩阵范数、特征值估计，数一考纲中以“Gerschgorin圆盘定理”和“瑞利商”为核心。 1. 行列式的放缩：对n阶行列式|A|，有（行和积），（列和积）； 2. 特征值的放缩（Gerschgorin圆盘定理）：矩阵A的任一特征值λ必落在某个圆盘|z − aii| ≤ ∑j ≠ i|aij|内，可用于估计特征值的范围； 3. 瑞利商放缩：对实对称矩阵A，其特征值满足（x ≠ 0），常用于特征值的上下界估计。

三、概率论与数理统计中的放缩技巧（核心是概率不等式）

四、放缩技巧的核心原则与避坑点

  1. 适度放缩：放缩的目的是“简化问题”，若放缩后两端极限/界不相等（如过度放大导致级数敛散性判断错误），则需调整放缩方式；
  2. 针对性放缩：根据问题类型选择技巧（如n项和用夹逼准则，概率估计用切比雪夫不等式）；
  3. 结合等价无穷小：放缩与等价无穷小替换可结合使用（如极限中先放缩再替换），简化复杂表达式。
