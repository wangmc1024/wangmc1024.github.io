极限型函数分类

发表于 2025-11-23 分类于 考研 阅读次数：

1. 含(_{n } x^{kn})的指数型极限函数（(k)为常数）

  • 核心特征：极限中含(x)的指数为(kn)（(n )），如(xn)、(x{2n})等形式。
  • 分类依据：指数函数(t^n)（(t = x^k)）的极限由底数(t)的绝对值与(1)的大小决定，因此按(|x|)与(1)的大小分类。
  • 解题思路：
    • (|x| < 1)：(_{n } (xk)n = 0)（底数绝对值小于(1)，指数无穷大时趋近于(0)）；
    • (|x| = 1)：(_{n } (xk)n = 1)（底数为()，指数无穷大时结果为(1)）；
    • (|x| > 1)：(_{n } (xk)n = )，分式形式需分子分母同除(x^{kn})化简。
  • 例子：求(f(x) = _{n } x)
    • (|x| < 1)：(x^{2n} )，故(f(x) = x)；
    • (|x| = 1)：(x^{2n} = 1)，故(f(x) = 0)；
    • (|x| > 1)：分子分母同除(x^{2n})，得(f(x) = -x)。

2. 含(_{x } e^{kx})的指数型极限函数（(k)为常数）

  • 核心特征：极限中含(e)的指数为(kx)（(x )），如(e{2x})、(e{-x})等形式。
  • 分类依据：指数函数(e^t)的性质（(t +)时(e^t +)，(t -)时(e^t )），按(k)的符号与(x)的趋向（(+/- )）分类。
  • 解题思路：
    • (k > 0)：(x +)时(e^{kx} +)，(x -)时(e^{kx} )；
    • (k < 0)：(x +)时(e^{kx} )，(x -)时(e^{kx} +)；
    • (k = 0)：(e^{kx} = 1)，极限为(1)。
  • 例子：求(_{x } )
    • (x +)：(e^{2x} +)，分子分母同除(e^{2x})，得极限为(1)；
    • (x -)：(e^{2x} )，得极限为(-1)；
    • 因此极限不存在（左右极限不等）。

3. 对数-幂函数型（({x ^+} x^k x)或({x +} )，(k)为常数）

  • 核心特征：幂函数与对数函数的乘积/商，属(0 )或()型，结果由幂函数指数(k)决定（对数增长远慢于幂函数）。
  • 分类依据：按幂函数指数(k)的符号（正、负、零）分类。
  • 解题思路：
    • 对(_{x ^+} x^k x)（(0 )型）：
      • (k > 0)：转化为()（()型），洛必达后极限为(0)；
      • (k )：(x^k )（(k < 0)）或(x^k = 1)（(k=0)），极限为(-)；
    • 对(_{x +} )（()型）：
      • (k > 0)：洛必达后极限为(0)；
      • (k )：极限为(+)。
  • 例子：求(_{x ^+} x^a x)（(a)为常数）
    • (a > 0)：洛必达后极限为(0)；
    • (a )：极限为(-)。
