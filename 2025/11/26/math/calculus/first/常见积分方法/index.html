常见积分方法详解：第一换元、第二换元与表格法

一、第一换元积分法（凑微分法）

第一换元积分法是最基础、最常用的积分方法，本质是复合函数求导（链式法则）的逆运算，通过“凑”出内层函数的微分，将复杂积分转化为基本积分公式的形式。

1. 核心原理

若已知 ( f(u)du = F(u) + C )，且 ( u = (x) ) 可导，则： [ f[(x)]‘(x)dx = f(u)du = F[(x)] + C ] 关键是识别被积函数中的“复合结构” ( f[(x)] ) 和“内层函数的导数” ( ’(x) )，并将 ( ’(x)dx ) 凑成 ( du )。

2. 适用场景

被积函数可拆分为复合函数与内层函数的导数的乘积，常见形式包括： - ( f(ax+b)dx )（如 ( (2x+1)dx )）； - ( x^n f(x^{n+1})dx )（如 ( xdx )）； - ( e^x f(e^x)dx )、( f(x)dx ) 等。

3. 解题步骤

  1. 观察结构：找出被积函数中的复合内层函数 ( u = (x) )；
  2. 凑微分：将 ( ’(x)dx ) 转化为 ( du )；
  3. 换元积分：代入 ( u ) 后用基本积分公式计算；
  4. 回代：将 ( u = (x) ) 代回结果。

4. 典型例子

计算 ( 2x(x^2)dx )： - 令 ( u = x^2 )，则 ( du = 2xdx )； - 原积分变为 ( u du = u + C )； - 回代得 ( (x^2) + C )。

二、第二换元积分法

当第一换元法难以凑出微分时，需通过主动设定变量代换 ( x = (t) ) 消去被积函数中的复杂结构（如根式、分式），将原积分转化为关于 ( t ) 的易解积分，本质是“反向换元”。

1. 核心原理

设 ( x = (t) ) 是单调、可导的函数，且 ( ‘(t) )，若 ( f[(t)]’(t)dt = F(t) + C )，则： [ f(x)dx = F[^{-1}(x)] + C ] 其中 ( t = ^{-1}(x) ) 是 ( x = (t) ) 的反函数。

三、表格法积分（分部积分的简化技巧）

表格法并非独立的积分方法，而是分部积分法的可视化简化，适用于被积函数为两个不同类型函数乘积的情况（如幂函数×指数函数、幂函数×三角函数、幂函数×对数函数等），可避免多次分部积分的繁琐计算。

1. 分部积分法的基础公式

若 ( u = u(x) )、( v = v(x) ) 可导，则： [ u dv = uv - v du ] 当需要多次应用分部积分时（如 ( x^n e^x dx )），表格法可通过“导数-积分表”快速推导结果。

2. 表格法的核心规则

  1. 列项：左侧列写 ( u ) 及其各阶导数，直到导数为0；右侧列写 ( dv ) 及其各次积分；
  2. 符号：第一行符号为正，后续行符号交替为负、正、负……；
  3. 计算：按“对角线相乘+符号”的规则累加，最后一行若导数未到0，需加余项 ( v_n du_n )（符号为最后一行的符号）。

3. 适用场景

被积函数为 ( P_n(x) f(x) )，其中 ( P_n(x) ) 是 ( n ) 次多项式，( f(x) ) 是易多次积分的函数（如 ( e^{ax} )、( bx )、( bx )）。

4. 典型例子

计算 ( x^2 e^x dx )： - 设 ( u = x^2 )（多项式，求导至0），( dv = e^x dx )（指数函数，积分）； - 制作表格： | 符号 | ( u ) 的导数 | ( dv ) 的积分 | |——|—————|—————-| | + | ( x^2 ) | ( e^x ) | | - | ( 2x ) | ( e^x ) | | + | ( 2 ) | ( e^x ) | | - | ( 0 ) | ( e^x ) | - 按规则计算：( +x^2 e^x - 2x e^x + 2e^x + C = ex(x2 - 2x + 2) + C )
