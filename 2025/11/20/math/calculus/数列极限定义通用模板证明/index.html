数列极限定义证明通用模板（ε-N三步法）

核心目标：对任意$\varepsilon > 0$，找到正整数$N$，当$n > N$时，$|x_n - A| < \varepsilon$（$A$为极限值）。

步骤 操作要点 典型处理方式
第一步：化简 对$ x_n - A
第二步：放缩 对化简后的$ x_n - A
第三步：取N 解“放大后的式子$< \varepsilon$”，得到$n > f(\varepsilon)$，取$N = \lfloor f(\varepsilon) \rfloor + 1$（向下取整+1保证$N$为正整数） 1. 若$\frac{2}{n^2} < \varepsilon$，则$n > \sqrt{\frac{2}{\varepsilon}}$，取$N = \lfloor \sqrt{\frac{2}{\varepsilon}} \rfloor + 1$；
2. 若$\frac{1}{10^n} < \varepsilon$，则$n > -\lg\varepsilon$，取$N = \lfloor -\lg\varepsilon \rfloor + 1$；
注：$f(\varepsilon)$只需存在，不唯一（如也可取$N = \lfloor \frac{1}{\varepsilon} \rfloor + 1$）。

模板套用验证（以$\lim\limits_{n\to\infty}\frac{n}{n+1}=1$为例）

  1. 化简：$|\frac{n}{n+1}-1|=|\frac{-1}{n+1}|=\frac{1}{n+1}$；

  2. 放缩：$\frac{1}{n+1} < \frac{1}{n}$；

  3. 取N：要使 $\frac{1}{n} < \varepsilon$ 解得 $n > \frac{1}{\varepsilon}$，取$N = \lfloor \frac{1}{\varepsilon} \rfloor + 1$，当$n > N$时，$|\frac{n}{n+1}-1| < \frac{1}{n} < \varepsilon$，得证。

考研易错点提醒

  1. 放缩不能“过度”：需保证放大后的式子仍能趋于0（不能把$\frac{1}{n}$放大为1，失去“任意小”的性质）；
    2.$N$必须是正整数：需用向下取整$+!1$，避免$N$为0/负数；

  2. 逻辑闭环：最后需明确“当$n > N$时，$|x_n - A| < \varepsilon$”，完成定义验证。
