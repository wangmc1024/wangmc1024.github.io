函数极限定义证明详细模板

一、$x \to x_0$ 型（$\varepsilon-\delta$ 定义）

定义本质

“当$x$无限接近$x_0$时，$f(x)$无限接近常数$A$”的严格数学表述为：对任意给定的正数$\varepsilon$（用于衡量$f(x)$与$A$的接近程度），总存在正数$\delta$（用于衡量$x$与$x_0$的接近程度），当$0 < |x - x_0| < \delta$时（此条件中$0 < |x - x_0|$是关键，用于排除$x = x_0$的情况——函数在$x_0$处可能无定义，不影响极限存在性），恒有$|f(x) - A| < \varepsilon$。

详细证明步骤（附示例：证明$\lim\limits_{x \to 1} \frac{x^2 - 1}{x - 1} = 2$）

  1. 化简$|f(x) - A|$
    对$|f(x) - A|$进行代数变形，常见手段包括因式分解、约分、通分等，核心是消去分母为0的项（因$x \neq x_0$，可放心约分），将其转化为仅含$|x - x_0|$的简单表达式。示例计算：先对分子因式分解，$x^2 - 1 = (x - 1)(x + 1)$，则
    $$\left| \frac{x^2 - 1}{x - 1} - 2 \right| = \left| \frac{(x - 1)(x + 1)}{x - 1} - 2 \right|$$
    约去$x - 1$（$x \neq 1$）后，化简为$|(x + 1) - 2| = |x - 1|$。

  2. 构造并求解$\delta$
    令化简后的$|f(x) - A| < \varepsilon$，得到关于$|x - x_0|$的不等式，解出$|x - x_0| <$ 仅含$\varepsilon$的式子，直接取$\delta$等于该式（也可取更小的正数，不影响逻辑严谨性）。示例计算：令$|x - 1| < \varepsilon$，直接取$\delta = \varepsilon$即可。

  3. 验证逻辑闭环
    严格推导“当$0 < |x - x_0| < \delta$时，$|f(x) - A| < \varepsilon$”，完成定义的“若-则”逻辑，证明“存在这样的$\delta$”。示例推导：当$0 < |x - 1| < \delta = \varepsilon$时，$\left| \frac{x^2 - 1}{x - 1} - 2 \right| = |x - 1| < \varepsilon$，满足$\varepsilon-\delta$定义，故极限成立。

常见易错点与应对方法

  1. 遗漏$0 < |x - x_0|$的条件：直接写$|x - x_0| < \delta$，忽略$x = x_0$时函数可能无定义的情况（如示例中$x=1$时分母为0）。应对：必须保留$0 < |x - x_0|$，明确排除$x = x_0$的情况。

  2. $\delta$中包含变量$x$：如错误取$\delta = \frac{\varepsilon}{x}$，违反“$\delta$仅由$\varepsilon$决定”的核心要求。应对：通过化简或放缩，确保最终得到的$\delta$是仅含$\varepsilon$的常数，与$x$无关。

二、$x \to \infty$ 型（$\varepsilon-X$ 定义，含$x \to +\infty$,$x \to -\infty$,$x \to \infty$）

定义本质

“当$x$的绝对值无限增大时，$f(x)$无限接近常数$A$”的严格数学表述分三类：

  1. $x \to +\infty$：对任意$\varepsilon > 0$，存在正数$X$，当$x > X$时，恒有$|f(x) - A| < \varepsilon$；

  2. $x \to -\infty$：对任意$\varepsilon > 0$，存在正数$X$，当$x < -X$时，恒有$|f(x) - A| < \varepsilon$；

  3. $x \to \infty$：对任意$\varepsilon > 0$，存在正数$X$，当$|x| > X$时，恒有$|f(x) - A| < \varepsilon$。

详细证明步骤（以$x \to +\infty$为例，附示例：证明$\lim\limits_{x \to +\infty} \frac{1}{x} = 0$）

  1. 放缩$|f(x) - A|$
    利用函数的有界性（如$|\sin x| \leq 1$、$|\cos x| \leq 1$）或不等式变形，将$|f(x) - A|$放大为仅含$x$的简单式子（如$\frac{C}{x}$、$\frac{C}{\sqrt{x}}$，$C$为常数），核心是让放大后的式子能通过“增大$x$”变得任意小。示例计算：直接化简得$\left| \frac{1}{x} - 0 \right| = \frac{1}{x}$，无需额外放缩（已是简单可解的形式）。

  2. 构造并求解$X$
    令放大后的$|f(x) - A| < \varepsilon$，解出关于$x$的不等式（如$x >$ 仅含$\varepsilon$的式子），取$X$等于该式（也可取更大的正数，保证逻辑严谨）。示例计算：令$\frac{1}{x} < \varepsilon$，两边取倒数（$x > 0$、$\varepsilon > 0$，不等号方向不变）得$x > \frac{1}{\varepsilon}$，取$X = \frac{1}{\varepsilon}$。

  3. 验证逻辑闭环
    严格推导“当$x > X$（或$x < -X$、$|x| > X$）时，$|f(x) - A| < \varepsilon$”，完成定义的逻辑验证。示例推导：当$x > X = \frac{1}{\varepsilon}$时，$\left| \frac{1}{x} - 0 \right| = \frac{1}{x} < \varepsilon$，满足$\varepsilon-X$定义，故极限成立。

不同趋近方向的调整方法

  1. $x \to -\infty$的调整：将步骤中的“$x > X$”改为“$x < -X$”，解不等式时注意绝对值的处理。如证明$\lim\limits_{x \to -\infty} \frac{1}{x} = 0$，先化简$|\frac{1}{x} - 0| = \frac{1}{|x|}$，令$\frac{1}{|x|} < \varepsilon$得$|x| > \frac{1}{\varepsilon}$，即$x < -\frac{1}{\varepsilon}$，取$X = \frac{1}{\varepsilon}$即可。

  2. $x \to \infty$的调整：将步骤中的“$x > X$”改为“$|x| > X$”，解不等式时直接针对$|x|$求解。如证明$\lim\limits_{x \to \infty} \frac{1}{x^2} = 0$，化简得$|\frac{1}{x^2} - 0| = \frac{1}{x2}$，令$\frac{1}{x2} < \varepsilon$得$|x| > \frac{1}{\sqrt{\varepsilon}}$，取$X = \frac{1}{\sqrt{\varepsilon}}$即可。

常见易错点与应对方法

  1. 放缩过度：如将$\frac{\sin x}{x}$错误放大为$\frac{x}{x} = 1$，导致式子无法小于任意小的$\varepsilon$。应对：放缩需“适度”，保留能趋于0的核心项，如利用$|\sin x| \leq 1$将$\frac{\sin x}{x}$放大为$\frac{1}{x}$，$\frac{1}{x}$可通过增大$x$变得任意小。

  2. $X$中包含变量$x$：如错误取$X = \frac{\varepsilon}{x}$，违反“$X$仅由$\varepsilon$决定”的要求。应对：放缩后确保解出的$X$是仅含$\varepsilon$的常数，与$x$无关。

三、综合示例（证明$\lim\limits_{x \to +\infty} \frac{\cos x}{\sqrt{x + 1}} = 0$）

  1. 放缩$|f(x) - A|$：利用$|\cos x| \leq 1$的有界性，化简得
    $$\left| \frac{\cos x}{\sqrt{x + 1}} - 0 \right| = \frac{|\cos x|}{\sqrt{x + 1}} \leq \frac{1}{\sqrt{x + 1}}$$
    为简化求解，进一步放缩：因$x + 1 > x$（$x > 0$），故$\sqrt{x + 1} > \sqrt{x}$，则$\frac{1}{\sqrt{x + 1}} < \frac{1}{\sqrt{x}}$。

  2. 构造并求解$X$：令$\frac{1}{\sqrt{x}} < \varepsilon$，两边平方（$x > 0$、$\varepsilon > 0$）得$x > \frac{1}{\varepsilon^2}$，取$X = \frac{1}{\varepsilon^2}$。

  3. 验证逻辑闭环：当$x > X = \frac{1}{\varepsilon^2}$时，
    $$\left| \frac{\cos x}{\sqrt{x + 1}} \right| \leq \frac{1}{\sqrt{x + 1}} < \frac{1}{\sqrt{x}} < \varepsilon$$
    满足$\varepsilon-X$定义，故$\lim\limits_{x \to +\infty} \frac{\cos x}{\sqrt{x + 1}} = 0$得证。

总结

函数极限定义证明的核心逻辑是**“转化-求解-验证”**：将复杂的$|f(x) - A|$通过化简或放缩，转化为仅含$|x - x_0|$（$x \to x_0$）或$x/|x|$（$x \to \infty$）的简单式子；通过解不等式找到仅由$\varepsilon$决定的$\delta$或$X$；最后验证“$x$满足距离条件时，$|f(x) - A| < \varepsilon$”，完成逻辑闭环。掌握这一核心逻辑，即可应对所有类型的函数极限定义证明题。
