简介

pybind11 是一个轻量级的仅头文件 C++ 库，用于将 C++ 代码暴露给 Python 调用，或者在 C++ 中调用 Python 代码。它比传统的 Boost.Python 更简洁高效，适用于现代 C++（C++11 及以上）。

核心特性

  • 函数绑定：将 C++ 函数暴露给 Python

  • 类绑定：将 C++ 类及其方法暴露给 Python

  • 数据类型转换：支持基本类型、STL 容器、NumPy 数组等

  • 异常处理：C++ 异常与 Python 异常的转换

  • 继承支持：支持 C++ 类继承和 Python 继承 C++ 类

  • 运算符重载：支持 C++ 运算符在 Python 中的使用

优势

  • 仅头文件：无需链接额外库，编译简单

  • 现代 C++：充分利用 C++11/14/17 特性

  • 自动类型推断：减少样板代码

  • 高性能：接近原生 C++ 性能

  • 易于使用：简洁的 API 设计

返回值策略

1. 策略概述

策略 描述
return_value_policy::take_ownership 接管现有对象的所有权，Python 会在引用计数为零时删除对象
return_value_policy::copy 创建返回对象的副本，由 Python 拥有
return_value_policy::move 使用 std::move 转移内容到新实例
return_value_policy::reference 引用现有对象但不接管所有权
return_value_policy::reference_internal 引用内部对象，确保父对象存活
return_value_policy::automatic 自动选择策略（默认）
return_value_policy::automatic_reference 类似 automatic，但对指针使用 reference 策略
