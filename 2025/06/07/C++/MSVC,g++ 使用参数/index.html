msvc和g++的常用参数

MSVC,g++ 使用参数

MSVC（Microsoft Visual C++）编译器核心为cl.exe（编译）和link.exe（链接），以下是常用参数精简说明：

一、编译参数（cl.exe）

  1. 基础控制

    • /c：仅编译生成.obj，不链接
    • /Fo<路径>：指定.obj输出路径（如/Foobj/）
    • /Fe<路径>：指定最终可执行文件路径

  2. 语言标准

    • /std:c++11/c++17/c++20：指定C++标准
    • /std:c11：指定C语言标准

  3. 宏与路径

    • /D<宏>：定义宏（如/DDEBUG）
    • /I<目录>：添加头文件搜索目录

  4. 警告与调试

    • /W4：最高警告级别；/WX：警告视为错误
    • /Zi：生成调试符号（.pdb）

  5. 优化

    • /Od：禁用优化（Debug用）
    • /O2：优化执行速度（Release默认）
    • /GL：启用全程序优化（需配合链接/LTCG）

二、链接参数（link.exe）

  1. 输出控制

    • /OUT:<路径>：指定输出文件（.exe/.dll）
    • /DLL：生成动态库（.dll）
    • /SUBSYSTEM:CONSOLE：控制台程序（默认）；WINDOWS：窗口程序

  2. 库与符号

    • /LIBPATH:<目录>：添加库文件搜索目录
    • <库名.lib>：指定依赖库（如kernel32.lib）
    • /DEBUG：生成调试符号（配合编译/Zi）

  3. 优化

    • /LTCG：链接时优化（配合编译/GL）
    • /OPT:REF：移除未使用代码

三、示例

  • Debug模式：
    cl /c /Od /Zi /W4 main.cpp /Foobj/
    link /DEBUG obj/main.obj /OUT:app.exe

  • Release模式：
    cl /c /O2 /GL /W4 /DRELEASE main.cpp
    link /LTCG main.obj /OUT:app.exe

通过VS项目属性（右键项目→属性）可图形化配置所有参数。

g++常用参数（简洁版）

g是GNU C编译器，常用参数如下：

  1. 基础控制

    • -c：仅编译生成.o目标文件，不链接
    • -o <文件名>：指定输出文件（如-o app.exe）
    • -I<目录>：添加头文件搜索路径

  2. 语言标准

    • -std=c++11/c++17/c++20：指定C++标准
    • -std=c11：指定C语言标准

  3. 宏与警告

    • -D<宏>：定义宏（如-DDEBUG）
    • -Wall：开启大部分警告；-Werror：警告视为错误

  4. 调试与优化

    • -g：生成调试信息（供gdb使用）
    • -O0：无优化（Debug）；-O2：中度优化（常用）；-O3：最高优化

  5. 链接控制

    • -L<目录>：添加库文件搜索路径
    • -l<库名>：链接指定库（如-lpthread链接线程库）
    • -shared：生成动态库（.so）

g++与MSVC参数对比整合表

功能类别 g++参数 MSVC参数（cl.exe/link.exe） 说明
输出指定 -o <文件> /Fe<文件>（cl）//OUT:<文件>（link） 指定最终输出文件
仅编译 -c /c（cl） 只生成目标文件，不链接
头文件路径 -I<目录> /I<目录>（cl） 添加头文件搜索目录
C++标准 -std=c++20 /std:c++20（cl） 指定C++标准版本
宏定义 -DDEBUG /DDEBUG（cl） 定义预处理宏
警告控制 -Wall -Werror /W4 /WX（cl） 开启警告并视为错误
调试信息 -g /Zi（cl）+/DEBUG（link） 生成调试符号
优化级别 -O0（无）/ -O2（中） /Od（无）/ /O2（中）（cl） 控制代码优化程度
库路径 -L<目录> /LIBPATH:<目录>（link） 添加库文件搜索目录
链接库 -l<库名>（如-lm） <库名.lib>（如math.lib）（link） 链接指定库文件
生成动态库 -shared /DLL（link） 生成动态链接库

示例对比：

  • g++编译Debug版本：
    g++ -c -g -O0 -Wall main.cpp -o obj/main.o
    g++ obj/main.o -o app.exe

  • MSVC编译Debug版本：
    cl /c /Zi /Od /W4 main.cpp /Foobj/main.obj
    link obj/main.obj /DEBUG /OUT:app.exe
