概率论；第一章

随机事件和概率 知识点总结

1. 随机事件与样本空间

1.1 基本概念

随机试验（E）：满足以下三个条件的试验：

  • 可以在相同条件下重复进行

  • 每次试验的可能结果不止一个，但能事先明确所有可能结果

  • 进行一次试验之前不能确定哪一个结果会出现

样本空间（Ω）：随机试验所有可能结果组成的集合

样本点（ω）：样本空间中的每个元素，即随机试验的每个可能结果

随机事件（A, B, C, …）：样本空间的子集，即由若干个样本点组成的集合

1.2 事件的分类

基本事件：由一个样本点组成的单点集

复合事件：由多个样本点组成的集合

必然事件：样本空间 Ω 本身，每次试验一定发生，记为 Ω

不可能事件：空集∅，每次试验一定不发生，记为∅

2. 事件的关系与运算

2.1 事件的关系

包含关系

  • 定义：若事件 A 发生必然导致事件 B 发生，则称 A 包含于 B，记为 A ⊂ B

  • 性质：

  • A ⊂ Ω

  • ∅ ⊂ A

  • 若 A ⊂ B 且 B ⊂ A，则 A = B

相等关系

  • 定义：若 A ⊂ B 且 B ⊂ A，则称 A 与 B 相等，记为 A = B

互斥关系（互不相容）

  • 定义：若事件 A 与 B 不能同时发生，即 A ∩ B = ∅，则称 A 与 B 互斥

  • 注意：互斥事件不一定是对立事件

对立关系（互逆）

  • 定义：若 A ∩ B = ∅且 A ∪ B = Ω，则称 A 与 B 互为对立事件，记为 B = (\overline{A})

  • 性质：

  • (\overline{\overline{A}} = A)

  • A 与(\overline{A})必互斥

  • P(A) + P((\overline{A})) = 1

2.2 事件的运算

并集（和事件）

  • 定义：事件 A 与 B 至少有一个发生，记为 A ∪ B 或 A + B

  • 性质：

  • A ∪ A = A

  • A ∪ Ω = Ω

  • A ∪ ∅ = A

  • 若 A ⊂ B，则 A ∪ B = B

交集（积事件）

  • 定义：事件 A 与 B 同时发生，记为 A ∩ B 或 AB

  • 性质：

  • A ∩ A = A

  • A ∩ Ω = A

  • A ∩ ∅ = ∅

  • 若 A ⊂ B，则 A ∩ B = A

差集

  • 定义：事件 A 发生而 B 不发生，记为 A - B

  • 等价关系：A - B = A(\overline{B}) = A - AB

对称差

  • 定义：事件 A 与 B 恰好有一个发生，记为 A Δ B

  • 等价关系：A Δ B = (A - B) ∪ (B - A) = (A ∪ B) - (A ∩ B)

2.3 运算律

交换律

  • A ∪ B = B ∪ A

  • A ∩ B = B ∩ A

结合律

  • (A ∪ B) ∪ C = A ∪ (B ∪ C)

  • (A ∩ B) ∩ C = A ∩ (B ∩ C)

分配律

  • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

  • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

对偶律（德摩根律）

  • (\overline{A ∪ B} = \overline{A} ∩ \overline{B})

  • (\overline{A ∩ B} = \overline{A} ∪ \overline{B})

  • 推广：(\overline{\bigcup_{i=1}^{n} A_i} = \bigcap_{i=1}^{n} \overline{A_i})，(\overline{\bigcap_{i=1}^{n} A_i} = \bigcup_{i=1}^{n} \overline{A_i})

3. 概率的定义与性质

3.1 概率的公理化定义

设 E 是随机试验，Ω 是样本空间，对于 E 的每个事件 A，赋予一个实数 P (A)，满足：

  1. 非负性：P(A) ≥ 0

  2. 规范性：P(Ω) = 1

  3. 可列可加性：对于两两互斥的事件 A₁, A₂, …，有

(P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i))

则称 P (A) 为事件 A 的概率。

3.2 概率的基本性质

  1. 不可能事件概率：P(∅) = 0

  2. 有限可加性：若 A₁, A₂, …, Aₙ两两互斥，则

(P\left(\bigcup_{i=1}^{n} A_i\right) = \sum_{i=1}^{n} P(A_i))

  1. 单调性：若 A ⊂ B，则 P (A) ≤ P (B)，且 P (B - A) = P (B) - P (A)

  2. 加法公式：P(A ∪ B) = P(A) + P(B) - P(AB)

  • 推广：P(A ∪ B ∪ C) = P(A) + P(B) + P© - P(AB) - P(AC) - P(BC) + P(ABC)

  1. 减法公式：P(A - B) = P(A) - P(AB) = P(A(\overline{B}))

  2. 对立事件概率：P((\overline{A})) = 1 - P(A)

4. 古典概型与几何概型

4.1 古典概型

定义

满足以下两个条件的概率模型：

  1. 样本空间中只有有限个基本事件

  2. 每个基本事件发生的可能性相等

计算公式

(P(A) = \frac{A包含的基本事件数}{基本事件总数} = \frac{m}{n})

适用场景

  • 摸球问题

  • 分房问题

  • 随机取数问题

  • 排列组合问题

典型例题

例 1：从 6 个圆中随机取出 2 个，已知相邻的圆都相切，求这 2 个圆不相切的概率。

解析：

  • 总取法：C₆² = 15 种

  • 相切的情况：7 种（数切点个数）

  • 不相切的概率：1 - 7/15 = 8/15

4.2 几何概型

定义

满足以下两个条件的概率模型：

  1. 样本空间是一个几何区域（长度、面积、体积等）

  2. 每个基本事件的发生与区域的几何度量成正比

计算公式

(P(A) = \frac{A的几何度量}{Ω的几何度量})

适用场景

  • 会面问题

  • 蒲丰投针问题

  • 随机变量取值问题

典型例题

例 2：甲乙两人约定在 6 点到 7 点之间会面，先到者等候 15 分钟，过时即离去。求两人能会面的概率。

解析：

  • 设甲到达时间为 x，乙到达时间为 y

  • 样本空间：Ω = {(x,y)|6 ≤ x ≤ 7, 6 ≤ y ≤ 7}

  • 会面条件：|x - y| ≤ 1/4

  • 概率：P = 1 - 2×(3/4×3/4)/2 = 7/16

5. 条件概率与五大公式

5.1 条件概率

定义

设 A, B 是两个事件，且 P (B) > 0，则在事件 B 发生的条件下事件 A 发生的概率为：

(P(A|B) = \frac{P(AB)}{P(B)})

性质

  1. P(A|Ω) = P(A)

  2. 若 A ⊂ B，则 P (A|B) = P (A)/P (B)

  3. 若 A 与 B 互斥，则 P (A|B) = 0

  4. P((\overline{A} )|B) = 1 - P(A|B)

5.2 乘法公式

(P(AB) = P(A|B)P(B) = P(B|A)P(A))

推广：

(P(A_1A_2…A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1A_2)…P(A_n|A_1A_2…A_{n-1}))

5.3 全概率公式

完备事件组

设 B₁, B₂, …, Bₙ是样本空间 Ω 的一个划分，即：

  1. Bᵢ ∩ Bⱼ = ∅ (i ≠ j)

  2. (\bigcup_{i=1}^{n} B_i = Ω)

  3. P(Bᵢ) > 0 (i = 1,2,…,n)

全概率公式

(P(A) = \sum_{i=1}^{n} P(A|B_i)P(B_i))

典型例题

例 3：三个箱子，第一个箱子有 4 黑 1 白，第二个有 3 黑 3 白，第三个有 3 黑 5 白。随机取一个箱子，再取一个球，求取得白球的概率。

解析：

  • P(A₁) = P(A₂) = P(A₃) = 1/3

  • P(B|A₁) = 1/5，P(B|A₂) = 3/6 = 1/2，P(B|A₃) = 5/8

  • P(B) = (1/3)(1/5) + (1/3)(1/2) + (1/3)(5/8) = 53/120

5.4 贝叶斯公式

(P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum_{j=1}^{n} P(A|B_j)P(B_j)})

意义

  • 先验概率：P (Bᵢ)

  • 后验概率：P (Bᵢ|A)

  • 用于由果溯因的推断

典型例题

例 4：某种疾病的发病率为 0.001，诊断试验的准确率为 0.95（患病者 95% 呈阳性，健康者 95% 呈阴性）。若某人试验呈阳性，求其实际患病的概率。

解析：

  • P© = 0.001，P((\overline{C} )) = 0.999

  • P(A|C) = 0.95，P(A|(\overline{C})) = 0.05

  • P(C|A) = (0.95×0.001)/(0.95×0.001 + 0.05×0.999) ≈ 0.0187

6. 事件的独立性

6.1 两个事件的独立性

定义

若 P (AB) = P (A) P (B)，则称事件 A 与 B 相互独立。

等价定义

  • P(A|B) = P(A) (P(B) > 0)

  • P(B|A) = P(B) (P(A) > 0)

性质

  1. 若 A 与 B 独立，则 A 与(\overline{B})，(\overline{A})与 B，(\overline{A})与(\overline{B})也相互独立

  2. 若 P (A) = 0 或 P (A) = 1，则 A 与任意事件独立

  3. 独立性与互斥性的关系：

  • 若 P (A) > 0，P (B) > 0，则 A 与 B 独立和互斥不能同时成立

  • 若 A 与 B 既独立又互斥，则 P (A) = 0 或 P (B) = 0

  1. (P(B|\overline{A}) + P(\overline{B}|\overline{A}) = 1 ) 则A和B独立

6.2 多个事件的独立性

三个事件的独立性

A, B, C 相互独立需要满足：

  1. P(AB) = P(A)P(B)

  2. P(AC) = P(A)P©

  3. P(BC) = P(B)P©

  4. P(ABC) = P(A)P(B)P©

性质

  • 若 A, B, C 相互独立，则任意两个事件也相互独立（两两独立）

  • 反之不成立，即两两独立不一定相互独立

6.3 n 重伯努利试验

定义

满足以下条件的试验：

  1. 每次试验只有两种可能结果：A 和(\overline{A})

  2. 每次试验中 P (A) = p，P ((\overline{A})) = 1-p

  3. 各次试验相互独立

  4. 共进行 n 次试验

二项概率公式

n 次试验中 A 恰好发生 k 次的概率：

(P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0,1,…,n)

典型例题

例 5：设每次试验成功的概率为 p，进行 3 次独立试验，在至少成功 1 次的条件下 3 次都成功的概率为 4/13，求 p。

解析：

  • P (3 次都成功) = p³

  • P (至少 1 次成功) = 1 - (1-p)³

  • 条件概率：p³ / [1 - (1-p)³] = 4/13

  • 解得：p = 2/3
