基本概念

批处理脚本（.bat 或 .cmd 文件）是Windows系统下的脚本文件，包含一系列DOS命令，用于自动化执行任务。它是一种解释性语言，由命令提示符（cmd.exe）解释执行。

特点：

  • 无需编译，直接执行

  • 逐行解释执行

  • 可以调用系统命令和其他程序

  • 支持变量、条件判断和循环等基本编程结构

常用命令

文件操作

  • dir - 显示目录内容

  • cd - 改变当前目录

  • md/mkdir - 创建目录

  • rd/rmdir - 删除目录

  • copy - 复制文件

  • move - 移动文件

  • del - 删除文件

  • ren/rename - 重命名文件

系统操作

  • echo - 显示文本或关闭/打开命令回显

  • cls - 清空屏幕

  • type - 显示文件内容

  • date - 显示或设置日期

  • time - 显示或设置时间

  • ver - 显示Windows版本

  • title - 设置命令提示符窗口标题

程序执行

  • start - 启动一个新窗口执行程序

  • call - 调用另一个批处理文件

  • exit - 退出批处理程序

  • pause - 暂停程序执行，等待用户按键

用户自定义变量

特殊参数扩展

根据官方文档，批处理脚本提供了强大的参数扩展功能，使用%~语法可以对参数进行各种转换。

基础扩展修饰符

语法 说明
%~1 展开%1并删除周围的引号
%~f1 将%1扩展到完全限定的路径名
%~d1 仅将%1扩展为驱动器号
%~p1 仅将%1扩展到路径
%~n1 仅将%1扩展到文件名
%~x1 仅将%1扩展到文件扩展名
%~s1 扩展路径只包含短名称
%~a1 将%1扩展到文件属性
%~t1 将%1扩展到文件的日期和时间
%~z1 将%1扩展到文件的大小
%~$PATH:1 搜索PATH环境变量中的目录，将%1扩展到找到的第一个文件的完整路径

%~dp0详解

%~dp0是最常用的特殊扩展之一：

  • d = 驱动器号

  • p = 路径

  • 0 = 表示批处理文件本身

特殊符号

重定向符号

  • > - 输出重定向（覆盖文件）

  • >> - 输出重定向（追加到文件）

  • < - 输入重定向

  • | - 管道符号，将前一个命令的输出作为后一个命令的输入

通配符

  • * - 匹配任意字符序列

  • ? - 匹配任意单个字符

其他符号

  • @ - 隐藏命令本身的显示

  • : - 标签定义

  • :: - 注释（与rem类似，但执行更快）

  • & - 命令分隔符，在同一行执行多个命令

  • && - 逻辑与，前一个命令成功才执行后一个

  • || - 逻辑或，前一个命令失败才执行后一个

  • () - 命令块，将多个命令作为一个整体执行

  • ^ - 转义字符，用于显示特殊字符本身
