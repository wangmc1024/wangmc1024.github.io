随机变量及其概率分布

随机变量的概念

定义

设E为随机试验，Ω为其样本空间，若对于每一个结果(样本点)ω ∈ Ω，都有一个实数X(ω)与之对应，则称定义在Ω上的实值函数X = X(ω)为随机变量。

分类

  • 离散型随机变量：取值为有限个或可列无限多个
  • 连续型随机变量：取值充满某个区间

常用性质

  1. 事件的等价表示：{X ∈ B} = {ω|X(ω) ∈ B}，其中B是实数集的子集
  2. 概率的基本性质：对于任意实数a < b，有P{X ≤ b} = P{X ≤ a} + P{a < X ≤ b}

离散型随机变量

分布律的定义

设离散型随机变量X可能的取值为x1, x2, …, xk, …，且P{X = xk} = pk,  k = 1, 2, 3, …，则称其为X的分布律。

分布律的性质

  1. pk ≥ 0,  k = 1, 2, …

常用离散型分布

1. 0-1分布

  • 分布律：P{X = 1} = p,  P{X = 0} = 1 − p
  • 记法：X ∼ B(1, p)
  • 期望：E(X) = p
  • 方差：D(X) = p(1 − p)
  • 常用性质：
    • 是二项分布的特例（n = 1）
    • 适用于任何只有两个结果的试验

2. 二项分布

  • 分布律：P{X = k} = Cnkpk(1 − p)n − k,  k = 0, 1, …, n
  • 记法：X ∼ B(n, p)
  • 期望：E(X) = np
  • 方差：D(X) = np(1 − p)
  • 常用性质：
    • 可加性：若X ∼ B(n, p), Y ∼ B(m, p)且独立，则X + Y ∼ B(n + m, p)
    • 最可能值：m = ⌊(n + 1)p⌋
    • 泊松近似：当n很大、p很小时，X ≈ P(np)

3. 泊松分布

  • 分布律：
  • 记法：X ∼ P(λ)
  • 期望：E(X) = λ
  • 方差：D(X) = λ
  • 常用性质：
    • 可加性：若X ∼ P(λ1), Y ∼ P(λ2)且独立，则X + Y ∼ P(λ1 + λ2)
    • 期望等于方差：E(X) = D(X) = λ
    • 稀有事件定律：适用于稀有事件的发生次数

4. 几何分布

  • 分布律：P{X = k} = (1 − p)k − 1p,  k = 1, 2, …

连续型随机变量

概率密度的定义

对于随机变量X的分布函数F(x)，如果存在非负可积函数f(x)，使得对于任意实数x有：

F(x) = ∫−∞xf(t)dt

则称X为连续型随机变量，f(x)称为X的概率密度函数。

概率密度的性质

  1. f(x) ≥ 0
  2. ∫−∞+∞f(x)dx = 1
  3. P{a < X ≤ b} = ∫abf(x)dx
  4. 对于任意实数a，P{X = a} = 0

分布函数

定义

设X是一个随机变量，x是任意实数，函数：

F(x) = P{X ≤ x},  x ∈ (−∞, +∞)

称为X的分布函数。

性质

  1. 单调性：F(x)是不减函数
  2. 有界性：0 ≤ F(x) ≤ 1，且F(−∞) = 0, F(+∞) = 1
  3. 右连续性：F(x + 0) = F(x)

常用性质

  1. 概率计算公式：
    • P{X ≤ b} = F(b)
    • P{a < X ≤ b} = F(b) − F(a)
    • P{X > b} = 1 − F(b)
    • P{X < b} = F(b − 0)
  2. 离散型与连续型的关系：
    • 离散型：F(x) = ∑xk ≤ xpk，在跳跃点处不连续
    • 连续型：F(x) = ∫−∞xf(t)dt，处处连续，且f(x) = F′(x)
  3. 分布函数的唯一性：分布函数唯一确定随机变量的分布

P{Y = yj} = ∑g(xi) = yjP{X = xi}

分布函数法

  1. 求Y = g(X)的分布函数：

FY(y) = P{Y ≤ y} = P{g(X) ≤ y}

数字特征总结

期望和方差的性质

期望的性质

  1. E(c) = c，其中c为常数
  2. E(cX) = cE(X)
  3. E(X + Y) = E(X) + E(Y)
  4. 若X和Y独立，则E(XY) = E(X)E(Y)

方差的性质

  1. D(c) = 0，其中c为常数
  2. D(cX) = c2D(X)
  3. D(X + Y) = D(X) + D(Y) + 2Cov(X, Y)
  4. 若X和Y独立，则D(X + Y) = D(X) + D(Y)

重难点总结

核心概念理解

  1. 随机变量的本质：函数而非变量，将随机试验结果数量化
  2. 分布函数的意义：完整描述随机变量的统计规律
  3. 概率密度的含义：概率的密度而非概率本身

重要性质和定理

  1. 泊松定理：二项分布的泊松近似条件和应用
  2. 指数分布的无记忆性：在可靠性理论中的重要应用
  3. 正态分布的性质：标准化、对称性、可加性
  4. 分布函数的性质：单调性、有界性、右连续性

计算技巧

  1. 离散型：准确列出所有可能取值，验证概率和为1
  2. 连续型：熟练掌握积分技巧，注意概率密度的非负性
  3. 函数分布：根据函数性质选择合适的计算方法
  4. 期望方差：利用性质简化计算，避免复杂的积分或求和

常见错误类型

  1. 概念混淆：离散型与连续型的区别，分布函数与概率密度的关系
  2. 计算错误：积分计算、导数计算、概率求和等
  3. 应用错误：选择不合适的分布模型，应用条件判断错误
  4. 公式记错：期望方差公式、变换公式等记忆错误
